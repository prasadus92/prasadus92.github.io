# Font Assets

These files are the Latin subsets used by the site and OG-image generator.

- `quicksand-latin.woff2`: Quicksand from Google Fonts
- `jetbrains-mono-latin.woff2`: JetBrains Mono from Google Fonts
- `quicksand-400.ttf`, `quicksand-500.ttf`, `quicksand-600.ttf`, `quicksand-700.ttf`: Quicksand TTF files used by the OG-image renderer

Both font families are distributed by Google Fonts under the SIL Open Font License 1.1.
